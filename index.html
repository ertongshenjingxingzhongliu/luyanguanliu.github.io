颅咽管瘤

1. 概述

颅咽管瘤是一种罕见的良性脑肿瘤，起源于颅咽管残余上皮细胞，多见于儿童和青少年。尽管是良性，但其位置靠近关键结构（如视神经、垂体、下丘脑），可能导致严重的神经和内分泌功能障碍。

2. 疾病分类

（1）按组织学分类

  • 成釉细胞型：多见于儿童，生长缓慢但易复发。
  • 乳头型：多见于成人，侵袭性较低。

（2）按位置分类

  • 鞍内型：位于蝶鞍内。
  • 鞍上型：位于蝶鞍上方，常压迫视神经和下丘脑。
  • 混合型：同时涉及鞍内和鞍上区域。

3. 病因

颅咽管瘤的具体病因尚不明确，可能与以下因素有关：

  • 胚胎发育异常：颅咽管残余上皮细胞异常增殖。
  • 遗传因素：某些基因突变可能增加发病风险，但尚未完全明确。

4. 临床症状

颅咽管瘤的临床症状主要包括：

（1）颅内压增高

  • 头痛：早晨加重，可能伴有恶心、呕吐。
  • 视乳头水肿：视力下降或视野缺损。

（2）视力障碍

  • 视力下降：肿瘤压迫视神经或视交叉。
  • 视野缺损：常见双颞侧偏盲。

（3）内分泌功能障碍

  • 生长迟缓：垂体功能受损导致。
  • 尿崩症：下丘脑受累引起。
  • 性早熟：下丘脑-垂体-性腺轴异常。

（4）其他症状

  • 精神行为异常：如情绪波动、记忆力下降。
  • 癫痫：少见，但可能发生。

5. 诊断依据

颅咽管瘤的诊断需结合临床表现、影像学检查和病理学检查：

（1）影像学检查

  • MRI：首选，显示肿瘤位置、大小及与周围结构的关系。
  • CT：辅助诊断，尤其对钙化灶敏感。

（2）内分泌检查

  • 激素水平检测：评估垂体功能，如生长激素、甲状腺激素等。

（3）病理学检查

  • 手术切除后病理诊断：明确肿瘤性质。

6. 治疗方法

颅咽管瘤的治疗需根据肿瘤大小、位置及患儿情况制定个体化方案，主要包括：

（1）手术治疗

  • 全切除：理想情况，但可能因肿瘤位置复杂而难以实现。
  • 部分切除：减少肿瘤体积，缓解症状。

（2）放射治疗

  • 术后放疗：用于残留肿瘤或复发病例。
  • 质子治疗：精准放疗，减少对周围组织的损伤。

（3）药物治疗

  • 激素替代治疗：针对内分泌功能障碍。
  • 靶向治疗：如BRAF抑制剂，适用于特定基因突变患者。

7. 预后

颅咽管瘤的预后因肿瘤位置、手术切除程度及治疗方法而异：

（1）预后因素

  • 手术切除程度：完全切除可显著改善预后。
  • 肿瘤位置：鞍上型预后较差。
  • 年龄：儿童患者预后优于成人。

（2）生存率

  • 5年生存率：总体较高，但复发率也较高。

（3）后遗症

  • 内分泌功能障碍：需长期激素替代治疗。
  • 视力障碍：部分患者可能遗留永久性视力损害。
  • 神经认知障碍：如记忆力下降、学习困难。
